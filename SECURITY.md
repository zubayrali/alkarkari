# Security Policy

## Supported versions

Security fixes are provided for the latest release on the `main` branch.

## Reporting a vulnerability

If you discover a security issue, please report it privately instead of opening a public issue.

Email: wangqiyangx@gmail.com

Include:

- A description of the issue
- Steps to reproduce
- Potential impact
- Any suggested fix, if you have one

You should receive a response within a reasonable time. Please do not disclose the issue publicly until it has been reviewed and addressed.
